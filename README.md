# windows-power-mode-switch

Windowsの電源モードをコマンドラインから簡単に切り替えることができるツールです。

## 機能

- 高パフォーマンス、省電力、バランスの3つの電源モードを切り替え可能
- 現在の電源モードを表示
- 日本語対応のインターフェース

## 使用方法

1. `PowerModeSwitch.bat`を**管理者権限**で実行してください
2. メニューから希望の電源モードを数字で選択します：
   - 1: 高パフォーマンス
   - 2: 省電力
   - 3: バランス
   - 4: 終了

## 必要要件

- Windows OS
- 管理者権限での実行

## 注意事項

- このスクリプトは管理者権限が必要です
- 電源モードの変更には数秒かかる場合があります
